# Library
C#, HTML, CSS, jQuery, Bootstrap, MSSQL Server gibi teknolojileri kullanarak web üzerinde yaptığım kütüphane projesinde; kitap ekleme, güncelleme ve silme gibi CRUD işlemleri bulunmaktadır. jQuery kullanarak arama kutucuğu yapılmıştır. Yazara ve yayınevine göre kütüphanede kaç kitap bulunduğu gösterilmiştir.

Projeyi çalıştırmak isteyenler:
1. Library.DataLayer içine gidin ve Context classına tıklayın. Server'ı SQL Server adınız ne ise onunla değiştirin.
2. Projeye migration ekleyin.
