# Library
Kendi kütüphanenizi kontrol edebileceğiniz, kitap ekleyip çıkarabileceğiniz, hangi yayınevinden ve hangi yazardan kaç tane olduğunu görebileceğiniz web tabanlı bir uygulama.

Projeyi çalıştırmak isteyenler:
1. Library.DataLayer içine gidin ve Context classına tıklayın. Server'ı SQL Server adınız ne ise onunla değiştirin.
2. Library.Datalayer katmanını "Set as Startup Project" olarak ayarlayıp Package Manager Console üzerinden  migration'ınızı oluşturun.
3. Library.UI kısmını tekrar "Set as Startup Project" olarak seçip projeyi çalıştırın.

Kullanılan Teknolojiler:
1. .NET
2. C#
3. HTML
4. CSS
5. jQuery
6. Bootstrap
7. MSSQL Server
8. ASP .NET Core
